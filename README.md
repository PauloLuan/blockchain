## 📝 Minimal Requirements

- NodeJs 14.x

## 🚀 Install

```sh
$ git clone https://github.com/PauloLuan/blockchain.git
```

```sh
$ yarn
```

### starting:

to start all apps in parallel:

```sh
$ yarn start
```

### Testing:

```sh
$ yarn test
```
